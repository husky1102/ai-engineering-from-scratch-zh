# 第 9 轮审查

日期：2026-06-09

范围：
- 复查第 8 轮后的翻译残留。
- 优先处理 LLM 工程阶段中基础概念密集、网页可见半翻译明显的课程。

## 结论

第 9 轮仍有问题，不能结束循环。

`phases/11-llm-engineering/05-context-engineering` 是当前最明确的问题课程。上下文工程本身是中文读者需要建立的新框架，但课程标题、导语、目标、图示、组件说明、动手步骤、术语表和 quiz 都大量保留普通英文，影响自然阅读。

## Findings

### P1 - Context Engineering 课程正文、图示、术语表和 quiz 半翻译

文件：
- `phases/11-llm-engineering/05-context-engineering/docs/zh-CN.md`
- `phases/11-llm-engineering/05-context-engineering/quiz.zh-CN.json`

证据：
- 标题和导语保留 `Context Engineering`、`Windows`、`Budgets`、`Memory`、`Retrieval`、`system instructions`、`retrieved documents`、`tool definitions`、`few-shot examples`。
- 学习目标和问题段保留 `context window components`、`token budgets`、`generation headroom`、`management strategies`、`sliding window`、`query type`、`window space`、`coding assistant breakdown`、`lost-in-the-middle effect`。
- Mermaid 图中大量可见标签仍为英文，例如 `Context Window`、`System Prompt`、`Tool Definitions`、`Attention Distribution Across Context`、`HIGH attention`、`Memory Architecture`、`Short-term Memory`、`Current Query`。
- 动手实现步骤保留 `Token Counter`、`Context Budget Manager`、`Lost-in-the-Middle Reordering`、`Conversation History Compressor`、`Dynamic Tool Selector`、`Full Context Assembly Pipeline`。
- 术语表表头和内容保留 `Term`、`What people say`、`What it actually means`、`Advanced prompt engineering`、`Loading stuff on the fly`、`Room for the answer`。
- quiz 中保留 `prompt engineering`、`context engineering`、`user query`、`retrieved docs`、`LLM performance`、`recency and primacy biases`、`budget management`、`generation headroom` 等可中文化内容。

影响：
- Context engineering 是 RAG、tool selection、memory 和 prompt caching 的共同底层概念；半英半中会让读者把“术语保留”和“未翻译”混在一起。

### P2 - quiz 题量和阶段分布不符合课程合同，网页只显示 5 题

文件：
- `phases/11-llm-engineering/05-context-engineering/quiz.json`
- `phases/11-llm-engineering/05-context-engineering/quiz.zh-CN.json`

证据：
- 本地网页复验时 `.quiz-section .quiz-question` 只有 5 个。
- 源文件只有 5 道题，并且阶段分布为 `pre=2/post=3`，缺少 `check` 阶段。
- 课程合同要求 6 道题：1 pre、3 check、2 post；站点也会按 `stage` 将 `check` 插入课中。

影响：
- 页面测验结构不完整，读者没有课中检查点；中文 quiz 即使翻译自然，也没有覆盖站点期望的学习节奏。

### P3 - 本课中文页面语言字段未反映 TypeScript 入口

文件：
- `phases/11-llm-engineering/05-context-engineering/docs/zh-CN.md`

证据：
- `code/` 中存在 `main.py` 和 `main.ts`，但中文文档语言字段只写 Python。

影响：
- 本轮先修复中文页面可见元数据，并在验证中同时运行 Python 与 TypeScript demo；不扩大到 README/ROADMAP 全局课程表。
