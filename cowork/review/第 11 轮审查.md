## Findings

### P1 - Python Environments 课程图示、text block、命令注释和术语表半翻译

文件：
- `phases/00-setup-and-tooling/06-python-environments/docs/zh-CN.md`

证据：
- Mermaid 图中可见标签保留 `Without virtual environments`、`System Python`、`Project A needs this`、`CONFLICT: only one torch version can exist`、`With virtual environments`。
- 目录策略 text block 保留 `shared lightweight env for phases 0-3`、`PyTorch env`、`same PyTorch env (symlink or shared)`、`might need different transformer versions`、`API SDKs, no torch needed`。
- 命令注释保留 `Install miniconda (not the full Anaconda)`、`base + PyTorch`、`everything`、`uv generates uv.lock automatically`、`pip-tools approach`、`BAD`、`GOOD`、`should show`、`These must be compatible`。
- 术语表仍有 `Lockfile`、`pyproject.toml`、`Package B depends on C` 等未自然中文化的普通解释。

影响：
- 读者在最基础的环境课中会看到术语保留和未翻译混在一起，降低可读性，也影响后续本地运行课程的信心。

### P2 - Python Environments 课程 quiz 只有 5 题且没有课中检查

文件：
- `phases/00-setup-and-tooling/06-python-environments/quiz.json`
- `phases/00-setup-and-tooling/06-python-environments/quiz.zh-CN.json`

影响：
