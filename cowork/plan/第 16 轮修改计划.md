# 第 16 轮 Dev Environment 修复实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复 Dev Environment 第一课的 demo 退出行为、旧 quiz 分布、代码测试缺口、入口命名和中文 Mermaid 标签。

**Architecture:** 本轮只改 `phases/00-setup-and-tooling/01-dev-environment` 下的 docs、quiz 和 code。默认运行保持教学诊断属性：打印失败检查但 exit 0；`--strict` 才把必需检查失败转为非零退出。新增 `main.py` / `main.ts` 作为稳定入口；保留原 `verify.py` / `verify.ts` 兼容文档语义。Python tests 使用 stdlib unittest，通过 monkeypatch 避免依赖本机实际环境。

**Tech Stack:** Markdown、JSON、Python stdlib、TypeScript Node stdlib、Rust stdlib、Python unittest、Python 校验脚本、Node.js 站点测试、Browser DOM 复验。
